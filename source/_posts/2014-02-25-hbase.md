## 下载安装hbase

```sh
$ tar xzvf hbase-0.96.1-hadoop2-bin.tar.gz
$ mv hbase-0.96.1-hadoop2 /usr/hbase
```

<!-- more -->

## 配置

### conf/hbase-env.sh

```sh
export JAVA_HOME=yourJavaHomePath
```

### hbase-site.xml

```xml
<configuration>
  <property>
    <name>hbase.rootdir</name>
    <value>hdfs://localhost:9000/hbase</value>
  </property>
  <property>
    <name>hbase.cluster.distributed</name>
    <value>true</value>
  </property>
  <property>
    <name>hbase.zookeeper.quorum</name>
    <value>localhost</value>
  </property>
  <property>
    <name>dfs.replication</name>
    <value>1</value>
  </property>
  <property>
    <name>hbase.zookeeper.property.clientPort</name>
    <value>2181</value>
  </property>
  <property>
    <name>hbase.zookeeper.property.dataDir</name>
    <value>yourDataDir</value>
  </property>
</configuration>
```

### 启动

```sh
$ start-dfs.sh
$ start-hbase.sh
$ jps
```

### Web查看状态

查看Master Status： `http://localhost:60010/master-status`

查看Region Server Status： `http://localhost:60030/rs-status`
